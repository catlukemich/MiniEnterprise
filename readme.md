# MiniEnterprise

About
-----

Mini enterprise is a simple management game, where player is about
to manage a company by creating new production chains.

Currently the game is a Work In Progress (WIP) at a pre - alpha stage, an almost early concept.

So far some features are implemented:
- The isometric view of the map
    - the tiles, trees, mountains, roads and some buildings (roughly 30 variants)
- Cars driving around (7 cars in 7 colors)
- Basic UI with 3 production chains:
    - coal, farming and oil

More features to be implemented are (somehow in order of priority, which might change):
- Building production chain buildings
- Some eyecandy (like visitation buildings)
- Trucks and rails
- Music and sound
- Multiplayer


Running
-------

In order to run the game - clone the repo and run the game.py script file, provided you have
pygame installed. There are no other requirements.

Or from command line, provided that the cloned directory is a working directory run:

` python3 game.py `
